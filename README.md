# 🏥 Liver Disease Prediction System

A Machine Learning-based web application that predicts the likelihood of liver disease using patient clinical data. The application is built with Python, Flask, and Scikit-learn and deployed on Render.

## 🚀 Live Demo
https://revolutionizing-liver-care-predicting-vlq5.onrender.com

## ✨ Features
- Predicts liver disease risk using patient health parameters
- Interactive web interface built with Flask
- Real-time prediction results
- Input validation and error handling
- Deployed as a live web application on Render

## 🛠️ Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- Joblib
- HTML/CSS
- Render

## 📊 Dataset
Indian Liver Patient Dataset (ILPD)

## 📥 Input Parameters
- Age
- Gender
- Total Bilirubin
- Direct Bilirubin
- Alkaline Phosphatase
- Alamine Aminotransferase (ALT)
- Aspartate Aminotransferase (AST)
- Total Proteins
- Albumin
- Albumin and Globulin Ratio

## 🤖 Machine Learning Model
- Algorithm: Random Forest Classifier
- Task: Binary Classification (Liver Disease / No Liver Disease)

## ⚠️ Disclaimer
This application is developed for educational and demonstration purposes only and should not be considered a medical diagnostic tool.
