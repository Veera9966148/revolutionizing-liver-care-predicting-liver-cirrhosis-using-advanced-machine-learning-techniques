# 🏥 Liver Cirrhosis Prediction System

## Overview
Machine Learning-based web application that predicts the likelihood of liver cirrhosis using patient clinical data.

## Features
- Patient data input form
- ML-based prediction
- Real-time results
- Simple user interface

## Technologies Used
- Python
- Scikit-learn
- Pandas
- Flask
- HTML/CSS

## Dataset
Indian Liver Patient Dataset (ILPD)

## Model
Random Forest Classifier
